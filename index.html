Der Palindrom Checker

Was ist ein Palindrom

Palindrom ist griechisch und bedeutet "rückwärts laufend". Es gibt Wortpalindrome und sogar Satzpalindrome. Palindrome müssen nicht immer einen Sinn ergeben, jedoch muss die Zeichenkette von vorne nach hinten sowie von hinten nach vorne gelesen übereinstimmen.

Geben Sie ein Wort/Zahl ein

Beispiele: racecar, 123321, abccba, Hannah.

Mein Code

In dem Code habe ich:

- .split("") benuzt um das Wort in die einzelnen Buchstaben zu trennen und jede Buchstabe in ein Array zu setzen.
- .reverse() benuzt um den ganzen Array umzudrehen.
- .join() um den ganzen Array zu nehmen und ihn zusammenzukleben.
Àm Ende habe ich das Wort mit dem umgedrehtem Wort vergliechen und geguckt ob die gleich sind.
- Wenn die gleich sind: taucht alert() auf und sagt, dass das Wort ein Palindrom ist.
- Wenn es nicht gleich ist, dann ist es kein Palindrome.


var istPalindrome = function(string) {
if (string === string.split('').reverse().join('')) {
alert(string + ' ist ein Palindrome.');
}
else {
alert(string + ' ist KEIN Palindrome');
}
}
document.getElementById('form1').onsubmit = function () {
istPalindrome(document.getElementById('inp1').value);
}
